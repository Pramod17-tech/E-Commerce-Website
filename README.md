# Fad Fusion E-commerce Website

A simple static online store built with HTML, CSS, and JavaScript.

## What this project includes

- `index.html` — homepage with featured products and customer reviews
- `products.html`, `products-men.html`, `products-women.html`, `products2.html`, `products2-men.html`, `products2-women.html` — product listings and category pages
- `cart.html` — shopping cart page with quantity updates and item removal
- `checkout.html` — basic checkout page
- `contact.html` — contact form with validation
- `about.html` — company/about page
- `script.js` — cart behavior, localStorage support, and form handling
- `style.css` — layout and visual styles using Bootstrap and custom CSS
- `images/` — product and hero images used by the site

## How to use

1. Open `index.html` in your browser.
2. Browse items and click **Add to Cart** to save products locally.
3. Visit `cart.html` to update quantities or remove items.
4. Complete the purchase flow by clicking **Checkout**.
5. Use `contact.html` to send a message using the contact form.

## Notes

- The cart uses browser `localStorage`, so items stay in the cart until cleared or the browser data is removed.
- This is a static demo site; it does not have a real payment gateway or server backend.

## Recommended improvement ideas
